## 2. 서비스 소개

뚝딱은 AI 영상 제작 전문가와 의뢰인을 연결하는 프리랜서 마켓플레이스입니다.

**핵심 가치:**
- AI 영상 분야(제품광고, 유튜브 편집, 모션그래픽, 숏폼, 아바타 등)에 특화
- 전문가 등급제 (일반/프라임/마스터)로 품질 보증
- 패키지형 가격 체계 (Basic / Standard / Premium)
- 견적 요청 → 비교 → 주문 → 납품 → 구매확정의 거래 사이클

**주요 이용자:**
| 구분 | 설명 |
|------|------|
| 의뢰인 (Client) | 영상 제작이 필요한 개인·기업·브랜드 |
| 전문가 (Expert) | AI 영상 제작 프리랜서·에이전시 |
| 관리자 (Admin) | 서비스 운영, 콘텐츠 심사, 정산 관리 |

| 1 | 월별 매출 차트 | 하드코딩 → DB `orders` 테이블 월별 집계 쿼리 |
| 2 | 대기 항목 카운트 | 서비스 승인, 전문가 승인, 신고, 환불, 문의 건수 실시간 |
| 3 | 이벤트 관리 CRUD | `events` 테이블 연동 (현재 static 배열) |
| 4 | 고객지원 티켓 | `support_tickets` + `support_messages` 테이블 연동 |

#### 3-2. 정산 시스템

| No. | 작업 | 설명 |
|-----|------|------|
| 1 | 수수료 정책 설정 | 플랫폼 수수료율 설정 (예: 15~20%) |
| 2 | 정산 주기 | 주간/격주/월간 정산 스케줄 설정 |
| 3 | 전문가 계좌 관리 | `expert_bank_accounts` 테이블 연동, 계좌 등록/변경 UI |
| 4 | 정산 처리 | 구매확정 주문의 수수료 차감 후 정산 금액 산출 |
| 5 | 정산 내역 | 전문가 대시보드 + 관리자 페이지에서 정산 이력 조회 |
| 6 | 실입금 처리 | (수동) 관리자가 은행 이체 후 정산 상태 변경 또는 (자동) 오픈뱅킹 API 연동 |

**정산 프로세스:**

```
구매 확정 → 정산 대기 (수수료 차감 계산)
    ↓
정산일 도래 → 정산 대상 집계
    ↓
입금 처리 (수동 이체 또는 자동)
    ↓
정산 완료 → 전문가에게 정산 완료 알림
```

| 1 | 상태 관리 | `services.status`: draft → pending_review → active / rejected |

| 3 | 자동 알림 | 승인/반려 시 전문가에게 인앱+이메일 알림 |
| 4 | 수정 후 재심사 | 반려된 서비스 수정 후 재제출 가능 |
